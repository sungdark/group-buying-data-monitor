# 团购+外卖实时自动数据监测系统解决方案

## 技术方案
基于 OpenClaw Skills 开发，采用模块化架构，各平台抓取能力独立封装，支持灵活调度与扩展。

### 核心组件
1. **抓取引擎**：基于 Playwright 实现反爬绕过，支持多平台动态渲染页面抓取
2. **调度器**：基于 cron 实现分时段调度，满足不同平台更新频率要求
3. **数据处理**：数据清洗、校验、去重，确保准确率98%+
4. **推送模块**：飞书机器人 WebHook 推送，支持自定义告警规则
5. **配置中心**：YAML 配置文件统一管理平台参数、调度规则、推送配置

### 平台支持
#### 评价平台（每2小时更新，10:00-20:00）
- ✅ 大众点评：评论数、评分、内容、门店排名
- ✅ 抖音来客：评论数、用户级别、评分、消费人数
- ✅ 高德地图：评论数、评分、回头率、榜单

#### 外卖平台（每30分钟更新，午餐10:30-12:30 / 晚餐17:00-19:00）
- ✅ 美团：订单量、转化率、商圈排名、门店评分
- ✅ 饿了么：订单量、转化率、商圈排名、门店评分
- ✅ 京东外卖：订单量、转化率、商圈排名、门店评分

## 部署说明
### 1. 环境安装
```bash
# 安装依赖
pip install -r requirements.txt

# 安装 Playwright 浏览器
playwright install chromium
```

### 2. 配置文件
复制 `config.example.yaml` 为 `config.yaml`，填写：
- 各平台账号/ Cookie
- 飞书机器人 WebHook 地址
- 调度规则配置
- 门店列表信息

### 3. 启动服务
```bash
# 启动监测服务
openclaw skill run data-monitoring
```

### 4. 验证
- 查看控制台输出确认任务正常调度
- 检查飞书群确认数据推送正常
- 查看 `logs/` 目录下日志排查问题
